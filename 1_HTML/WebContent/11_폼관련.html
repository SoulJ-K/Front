폼 관련 태그

form태그는 html에서 사용자가 입력할 수 있는 양식 제공
form태그 안의 input태그들을 통해 사용자가 입력한 정보를 서버로 넘김
* action 속성 : 폼에 입력된 값들을 전송받을 서버의 클래스 명 입력
* method 속성 : get/post방식으로 전송 방식 지정

검색할 내용 :

submit버튼을 눌렀을 때 method에 지정한 방식으로 action에 지정한 경로에 input태그의 값을 전달함

필드셋의 제목을 작성하는 부분

- 한 줄 짜리 텍스트를 입력할 수 있는 텍스트 상자


type="password"

- 비밀번호를 입력할 수 있는 텍스트 상자

숫자와 관련된 input태그 속성들

type="number"

- 입력 창에 숫자를 입력하는 것이 기본이지만, 브라우저에 따라 스핀박스가 표시되기도함

type="range"

- 슬라이드바를 통해 숫자 지정 가능
